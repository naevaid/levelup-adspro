# Web App

Frontend `LevelUP adsPRO` berbasis `Next.js`.

## Menjalankan Lokal

Dari root repo:

```bash
npm run dev:web
```

Atau dari folder ini:

```bash
npm run dev
```

App akan berjalan di `http://localhost:3000`.

## Environment

Salin template berikut sebelum development lokal:

```bash
copy .env.example .env.local
```

Variable awal:

- `NEXT_PUBLIC_API_BASE_URL`
- `NEXT_PUBLIC_APP_ENV`
